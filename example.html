Parameter
参数名参数值(默认值可省略)
typescrollX (默认) : 水平滚动
scrollY : 垂直滚动
fade : 淡入淡出
fragment : 碎片翻页
srcArray : 传入路径数组
indicatorBoolean : 是否开启下方指示器(默认false)
arrowBoolean : 是否开启前后翻页按钮(默认false)
speedNumber : 翻页动画速度(默认600)
autoPlayBoolean : 是否开启自动轮播(默认true)
playSpeedNumber : 自动播放间隔时间(默认3000毫秒)
fragmentNumNumber : 碎片翻页中碎片个数(默认4个)
colorString : 指示器未选中颜色(默认'white')
activeColorString : 指示器选中颜色(默认'#248BD6')
